# 🧭 FellowFinder Light

**专注超视距队友感知的超轻量模组** —— 即使朋友在视野之外，也能感知彼此方位。

[![GitHub Release](https://img.shields.io/badge/Download-v1.0.0-brightgreen)](https://github.com/L0stray/fellowfinder-light/releases/tag/v1.0.0)

---

## ✨ 功能

- 🧭 **罗盘条 HUD**：屏幕顶部显示其他玩家的方位和水平距离
- 🎨 **UUID 固定颜色**：每个玩家拥有独特的颜色，一眼辨认
- 👻 **遵循原版规则**：潜行、隐身、旁观者、戴着头都会自动隐藏
- 📡 **超视距广播**：房主安装即自动开启，无需配置
- 🪶 **极致轻量**：仅 18.8 KB，极低依赖
- ⌨️ **F1 一键隐藏**：截图时自动跟随原版 HUD 隐藏

---

## 📥 安装方法

1. 在 [Releases](https://github.com/L0stray/fellowfinder-light/releases) 页面下载对应版本的 `.jar` 文件
2. 放入 `.minecraft/mods/` 文件夹
3. 确保已安装 **Fabric Loader**（>=0.15.0）和 **Fabric API**
4. 启动游戏，开箱即用
